# Lets talk about - API (of shyftplan)

[View Slides](https://api.lets-talk-about.tamino.dev)

A slideshow introducing the shyftplan API — what it is, why it exists, how data is structured, and how to interact with it. Covers REST concepts (HTTP methods, URLs, parameters), authentication, core data models (User, Employment, Shift, Evaluation, etc.), and a live API playground demo.

Built with [lets-talk-about](https://github.com/tamino-martinius/lets-talk-about) ([npm](https://www.npmjs.com/package/lets-talk-about)) — a tool for creating slideshows from Markdown.
